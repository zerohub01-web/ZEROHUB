# ZERO OS Monorepo

## Apps
- apps/web: Next.js 14 + TypeScript public + admin UI
- apps/api: Express + MongoDB API, auth, analytics, automation

## Setup
1. Copy `.env.example` files from each app and fill values.
2. `npm install`
3. `npm run dev`

## Hidden Admin Route
- `http://localhost:3000/zero-control`
